📖 Developer Resources

Complete API Reference, Best Practices & Migration Guide

Complete reference for all funcwc APIs, including function signatures, parameters, and return values.
⚙️

defineComponent

Core API for creating components with modern function-style props and CSS-only format.

defineComponent("component-name", { render: ({ /* function-style props */ }) => JSX, styles: { /* CSS-only format */ }, api: { /* HTMX endpoints */ } });
⚛️

defineReactiveComponent

Enhanced component API with hybrid reactivity system for complex state management.

defineReactiveComponent("reactive-name", { cssReactions: { /* CSS property reactivity */ }, stateSubscriptions: { /* Pub/sub state */ }, eventListeners: { /* DOM events */ }, onMount: "/* initialization code */" });
