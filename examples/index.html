🚀 Unified Styles API

All examples now use the revolutionary unified styles approach - no more duplication between classes and styles!

Zero Duplication

Define class names and their styles together in one place. The old approach required defining class names twice - once in classes and again in styles.

✨ View New Unified Styles API 
// 🎉 NEW: Ultra-Simplified CSS-Only Format!
defineComponent("smart-counter", {
  props: (attrs) => ({
    initialCount: parseInt(attrs.initialCount || "0"),
    step: parseInt(attrs.step || "1")
  }),
  styles: {
    // ✨ Just CSS properties - class names auto-generated!
    container: \`{ display: flex; gap: 0.5rem; border: 2px solid #007bff; }\`,
    button: \`{ background: #007bff; color: white; padding: 0.5rem; }\`,
    buttonHover: \`{ background: #0056b3; }\`, // → .button-hover
    display: \`{ font-size: 1.5rem; font-weight: bold; }\`
  },
  render: ({ initialCount, step }, api, classes) => (
    <div class={classes.container}> {/* .container */}
      <button class={classes.button}>-{step}</button> {/* .button */}
      <span class={classes.display}>{initialCount}</span> {/* .display */}
      <button class={classes.button}>+{step}</button>
    </div>
  )
});
